# wee-frame

## 简介

个人项目开发专用React上层封装框架

## 使用文档

### 初始化项目

1. `npm create @vvee/app`
2. `npm start`

## 开发指引

wee-frame采用lerna多包管理:

- app: 应用模板，用于生成应用项目
- cli: 框架的命令行工具，用于初始化项目，启动项目，构建项目等
- core: 框架的核心，提供框架的基础能力

开发环境依赖：`>= Node 16`

1. 根目录下，执行`npm install`
2. 切换到各个packages中去开发
